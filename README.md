
# nok.js

Extremely basic, and <b>not</b> production ready bar chart. Used for my experiments.

```
nok(array, canvasID, color, width, height)

nok([1,2,3,4,5], "MyCanvas", "black")
nok(MyArrayVariable, "MyCanvas", "rgba(0,0,0,0.7)", 1000, 200)
```
